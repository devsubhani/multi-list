multi-list

A jQuery plugin to turn an unordered list into a multiple selectable list. Source code available on Github.

Turns an unordered list like this one:

  • Europe
  • Oceania
  • Africa
  • Asia
  • North America
  • South America
  • Middle East

into a list with multiple selection options and a filter:

  • Europe
  • Oceania
  • Africa
  • Asia
  • North America
  • South America
  • Middle East

Multiple options and methods allowed!
